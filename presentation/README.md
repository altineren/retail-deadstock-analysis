# Presentation Build Plan

1. **Template Selection**
   - Canva "Tech Startup Pitch" or similar; ensure 16:9 ratio, consistent palette.
2. **Slide Breakdown (max 10 slides)**
   1. Title: team name/logo, project title, members.
   2. Problem definition: dead stock definition + pain metrics (visual chart).
   3. Impact slide: operational + CX consequences (icons for clarity).
   4. AI solution overview: pipeline graphic, bullet describing prediction use case.
   5. Architecture & tech stack: diagram snippet from report.
   6. Data & model: what data, model type, expected metrics.
   7. Demo showcase: mockup screenshots annotated.
   8. Business impact: ROI table, savings estimates.
   9. Implementation roadmap: 3-phase plan.
   10. Conclusion + next steps.
3. **Design Assets**
   - Architecture diagram export (PNG)
   - Mockup screenshots (Figma/Canva)
   - Icons for warehouse, AI, ROI
4. **Speaking Cues**
   - 30 sec per slide target; rehearse to stay under 5 minutes.
5. **To-Do**
   - Draft storyline after report outline approved.
   - Collect metrics for impact slide.
   - Insert demo screenshots once prototype ready.
